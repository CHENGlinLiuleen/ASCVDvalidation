# ASCVDvlidation
## Overview
This repository contains all the code associated with the article: "Gamma-Glutamyl Transferase Enhances Hard ASCVD Risk Prediction: Evidence from NHANES and CHARLS with Machine Learning Validation." The code comprehensively covers the majority of machine learning and visualization content used in the study. It can be executed in R version 4.5.1 or R 4.4.3.

## Explanation
### 2.1. FilteredData
This folder contains sample information from NHANES and CHARLS, along with the data used at each step of the Results section.

### 2.2. ResultsQR
This folder provides a quick reference for the results generated by the machine learning models, facilitating easier code reproduction. The original code for these results is available in Part 3 of the repository.

### 2.3. Sor.R
This script contains a rewritten function used in the machine learning process.

## Additional Notes
The code and data for calculating AHA risk can be obtained by contacting the corresponding author. After the article is published, a demo using a specific NHANES cycle will be provided for reference.
